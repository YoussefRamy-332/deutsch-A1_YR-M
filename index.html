Imperativ

We use This Rule with (Du -Ihr -Sie) with any verb

EX: Kommen
    Du
    1- We cross (St) kommst =komm
    2- We but (!) komm!
    3- Komm!



    Ihr
    1- We cross (en) kommen =komm
    2- We but (t) komm =kommt
    3- We but (!) kommt !

    Sie
    1- We but (en) kommen
    2- We but (Sie) kommen Sie
    3- We but (!) kommen Sie !
